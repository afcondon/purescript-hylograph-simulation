High-Level API Test

runSimulation: One config record, two engines

The Simplified API: 
handle <- runSimulation
  { engine: D3  -- or WASM (same config!)
  , setup: setup "physics" [manyBody "charge", center "center"]
  , nodes: myNodes
  , links: []
  , container: "#my-svg"
  , nodeTemplate: \_ -> elem Circle [...]
  , onComplete: log "done"
  , alphaMin: 0.001
  }
That's it! No coordinator setup, no consumer registration, no manual rendering.
